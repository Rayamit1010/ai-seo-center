# Feature Catalog

## Core Product Areas

### Dashboard
- SEO and AI operations overview
- backlink pipeline KPIs
- provider health and failover visibility
- recent automation and incident signals

### AI Chat
- multi-provider routing across Claude, ChatGPT, Gemini, Grok, and Groq
- smart auto routing and manual provider selection
- compare mode across multiple models
- project-memory-aware answers
- streaming responses with saved sessions

### Projects
- per-website strategy memory
- brand voice, business goals, and conversion targets
- backlink rules and content playbooks
- niche positioning and competitor notes
- Search Console and GA4 property mapping per website
- encrypted CMS publishing credentials per website
- per-project connection testing for Google and CMS integrations

### Backlink Agent
- campaign creation and lifecycle controls
- automated discovery, qualification, contact finding, drafting, sending, and tracking
- spam-risk and quality signals
- agent run history and live funnel stats
- manual "Run Now" control plus background automation cadence

### Reports
- report generation from completed audits
- branded client-ready documents
- recurring daily, weekly, and monthly schedules
- draft or email delivery workflows
- delivery logs and schedule management
- live Search Console and GA4 deltas embedded into saved reports when connected

### Ops Center
- queue visibility
- processing, pending, completed, and dead-letter job visibility
- recent background incidents
- AI provider problems in human-readable form

### AI Analytics
- provider attempts, failures, failovers, and latency
- health scoring by provider
- telemetry-backed routing insight

### Content Operations
- on-page content analysis
- meta tag generation
- schema generation
- direct publishing to connected WordPress or webhook CMS targets

## Platform Features

- NextAuth credentials authentication
- encrypted provider API key storage
- queue-backed background jobs
- database-backed durable job processing with retries and dead-letter support
- database lease protection for recurring jobs and agent-cycle claims
- Postgres-ready Prisma schema sync workflow for production
- Prisma persistence layer
- human-friendly error messaging
- protected per-user data boundaries
